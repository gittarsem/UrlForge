# URLForge

Production-ready URL shortening platform built with Spring Boot and React.

## Tech Stack

### Backend

- Java 21
- Spring Boot
- PostgreSQL
- Redis
- JWT
- Docker

### Frontend

- React
- TailwindCSS

## Features

- URL Shortening
- QR Code Generation
- Analytics
- JWT Authentication
- Rate Limiting
- Docker
- Swagger
